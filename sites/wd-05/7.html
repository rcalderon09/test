Make Your Mac Run Like New Again

Powerful, efficient and extremely versatile, Tidy Up by Hyperbolic is the complete duplicate finder and disk tidiness utility software designed to help users clean up their Mac hard drives. Tidy Up doesn’t waste time or computer resources by indexing all the contents of your hard drive, it simply searches by creator, application, content, type, time created or modified, extension, name, label, visibility, or date/age of the file. With a new user interface, priority option searching, multi-tasking features and more, Tidy Up gets your sluggish Mac running like new again.
